This HTML files is used to design the hotspots on the Streetcar map. Use a tool to get the hotspots in the correct place, use the 'convert.sh' tool to create the actual code tables in StreetcarHotSpotTable.txt

  • Replace <area shape="rect" coords=" with HS_RECT(
  • Replace " href="with ,
  • Replace /> with )

The Xcode project will make a text file called StreetcarHotSpotTable.txt and include it into RailMapsView.m automatically.

Note that there is a file called StreetcarHotSpotTiles.txt which is also generated but it is not automatic, you have to execute a special "test case" called test_generate_HotSpot_tiles to generate that code.
