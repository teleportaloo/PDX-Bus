This HTML files is used to design the hotspots on the TriMet rail map. Use a tool to get the hotspots in the correct place, use the 'convert.sh' tool to create the actual code tables in MaxHotSpotTable.txt

  • Replace <area shape="poly" coords=" with HS_START_POLY
  • Replace " href= with HS_END_POLY
  • Replace /> with HS_END

The Xcode project will make a text file called MaxHotSpotTable.txt and include it into RailMapsView.m automatically.

Note that there is a file called MaxHotSpotTiles.c which is also generated but it is not automatic, you have to execute a special "test case" called test_generate_HotSpot_tiles to generate that code.
